工具盒配置教程

发表于2026-07-05|更新于2026-07-13|教程
|总字数:747|阅读时长:3分钟|浏览量:

工具盒是一个可自定义的导航页面，支持站点收藏、搜索引擎切换、分类管理。所有配置通过两个 YAML 数据文件完成，无需修改代码。

文件位置

导航配置 (navi.yml)

分组分类

分类下包含多个子分类，每个子分类下再列站点：

添加新站点

图标可直接省略 icon 字段，工具盒会自动使用 Google Favicons 服务获取网站图标。

搜索引擎配置 (navi_search.yml)

搜索引擎分为多个分组，每个分组内可定义多个引擎：

字段说明：

字段 说明
defaultEngine 默认搜索引擎 URL
groups[].name 分组名（顶部标签）
groups[].id 分组标识（唯一，单个字母即可）
groups[].icon 标签图标（Font Awesome 类名）
groups[].color 标签颜色（bg-blue / bg-teal / bg-pink / bg-yl / bg-gj / bg-tp / bg-sq / bg-sc / bg-red）
items[].name 引擎名称
items[].url 搜索 URL（关键词部分用空字符串，前端会拼接）
items[].ph 搜索框 placeholder 提示文字
items[].style 引擎名称文字颜色样式
