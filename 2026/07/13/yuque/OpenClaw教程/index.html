介绍

官网地址：https://clawd.org.cn/

安装部署

windows

Node.js安装

  • 访问Node.js官网 https://nodejs.org/

  • 运行安装包
  • 勾选“Automatically install the necessary tools”
    • 以便自动安装编译工具
  • 安装完成后，管理员身份打开PowerShell，验证安装
  • 配置npm国内镜像-加速下载

OpenClaw安装

  • 执行安装命令
  • 验证是否安装成功
  • 配置向导

linux

  1. 更新系统包列表
  1. 安装编译工具和基础依赖（Node 编译必需）
  1. 安装openclaw
  • 脚本会自动完成：
    • 检测并安装 Node 22+
    • 全局安装 OpenClaw CLI
    • 启动引导配置向导（脚本执行完成后会进入引导配置，先ctrl+c退出，稍后进入配置）

卸载

windows

打开终端（CMD或者PowerShell）

Skill插件

Skill介绍

Skill插件

  • Skill定义：一个Skill是一个Node.js模块，导出符合规范的对象
  • Skill结构：
    • name：技能名称，唯一标识。
    • description：技能描述，用于模型选择。
    • parameters：参数定义（JSON Schema格式），说明技能需要哪些输入。
    • execute：核心执行函数，接收参数，返回结果。
  • Skill加载机制：
    • 启动时扫描指定目录（内置、本地托管、工作区）加载Skill。
    • 支持热加载：开发环境下修改Skill代码后自动重载。
  • Skill权限管理：每个Skill可以声明所需权限（如文件读写、网络访问），在配置中控制是否授予。

Skill如何让AI“会做事”

  • 传统AI对话机器人只能输出文本，无法操作实际系统。
  • OpenClaw通过Skill将AI的“想法”转化为“行动”：
    • 模型输出调用某个Skill及其参数
    • 系统执行Skill并返回结果
    • 模型再根据结果回复用户

Skill使用

clawhub（或国内镜像 skillhub）来安装社区分享的技能

  • 官网：https://clawhub.ai/

在openclaw使用对话安装skill

使用命令安装

  • 安装clawhub cli
  • 通过token登录clawhub cli
  • 安装skill:Tavliy-search(联网搜索)
  • 卸载Skill

opencode的官网：https://opencode.ai/zen

OpenCode Zen的API地址是：https://opencode.ai/zen/v1

openclaw配置：

  • 在命令行中启动配置向导
  • 配置opencode-zen
  • 免费的model

问题记录

为什么会报这个错误：

  • 这是因为 OpenClaw 在较新的版本中加强了安全限制。浏览器在非安全上下文（即 http://公网IP）下会硬性禁用 WebCrypto API，而 OpenClaw 恰恰依赖这个 API 来生成设备身份。因此，仅仅设置 allowInsecureAuth: true 已经无法绕过这个浏览器底层的限制，必须同时禁用设备身份验证才能生效。

解决方案：

  • 补充禁用设备身份验证配置
  • 重启服务
